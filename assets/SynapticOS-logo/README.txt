SynapticOS — Logo & Brand Assets
================================

CONCEPT
A microchip (the embedded silicon / OS) with a synaptic signal firing across
the cleft at its centre — the bright core is the active synapse. It encodes
exactly what SynapticOS is: an AI-native, inference-first RTOS for edge silicon.
The signal also reads as an "S".

COLOURS
  Electric blue   #2F6BFF
  Cyan            #17C9C0
  Spark           #9FFBEE   (synapse glow)
  Spark core      #E9FFFB   (firing point)
  Slate (neutral) #1E2327   (matches the repo theme colour; use on light bg)
  Near-white      #F5F7FA   (wordmark on dark bg)

TYPEFACE
  Wordmark is IBM Plex Sans SemiBold (OFL). "OS" uses the blue→cyan gradient.
  All text in the SVGs is converted to vector paths, so the files render
  identically without the font installed.

WHICH FILE TO USE
  LinkedIn page logo (avatar)  -> png/icon/synapticos-icon-badge-slate-400.png
                                  (or -white- for a light avatar)
  GitHub README header         -> svg/synapticos-horizontal-light.svg  (light READMEs)
                                  svg/synapticos-horizontal-dark.svg   (dark READMEs)
                                  GitHub renders SVG; for a raster fallback use the
                                  matching png/horizontal/*-1592w.png (2x).
  Favicon / small icon         -> png/icon/synapticos-icon-32.png (transparent)
  Print / single-colour        -> svg/png mono versions
  Vertical / square layouts    -> stacked lockups

FILES
  svg/    Scalable masters (preferred wherever SVG is supported)
  png/icon/        transparent mark (512/256/128/64/32) + on-background badges
  png/horizontal/  1x and 2x, light & dark
  png/stacked/     1x and 2x, light & dark
  png/mono/        single-colour (slate / white)

CLEAR SPACE & MIN SIZE
  Keep clear space around the logo equal to one chip-pin length.
  Don't recolour the mark, stretch it, add effects, or place the colour mark on
  a busy background — use the mono or badge versions instead.
  Minimum icon size: 24 px. Below ~20 px the synapse detail starts to close up.
